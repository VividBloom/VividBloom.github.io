HarmonyOS开发-ArkTS基础语法二

运算符、空安全机制和流程控制语句

运算符

赋值运算符

  • 基础赋值: =

  • 复合赋值: +=, -=, *=, /=, %=, <<=, >>=, >>>=, &=, |=, ^=

比较运算符

  • 严格比较: ===(值与类型均相等), !==(值与类型均不等)

  • 宽松比较: ==, !=, >, <, >=, <=

    优先使用严格比较, 避免隐式类型转换问题

算术运算符

  • 一元运算符: +, -, ++, --
  • 二元运算符: +, -, *, /, %

逻辑运算符

  • &&: 逻辑与, 所有条件为真时返回true

  • ||: 逻辑或, 任一条件为真时返回true

  • !: 逻辑非, 布尔值取反

    逻辑与短路特性: 若第一个条件为false, 直接路过后续条件

    逻辑与短路特性: 若第一个条件为true, 直接路过后续条件

位运算符

  • 位运算是直接对整数的二进制位进制操作的运算方式

  • 基础运算符: &, |, ^, ~

  • 位移运算符: <<, >>, >>>

    按位与(&): 两个操作数的对应二进制位均为1时, 结果为1, 否则为0

空安全

一般来说, 有时会存在声明变量时不确定初始值, 在这类情况下, 通常使用联合类型包含null值或undefined, null表示显示赋值的空值, 常用于主动清空对象引用, undefined表示变量未初始化或属性不存在.

使用时, 不对可为null值的变量进制非空校验, 那么在编译或者运行时就会发生错误, 无法正确运行.

  1. 使用if/else进制判空

  2. 使用?.可选链, 如果是null, 运算符会返回undefined

  3. 使用非空断言运算符, !断言左边的值非空

    在确认对象为非空才能使用

  4. 使用空值合并运算符, ??左边的值为null时会返回表达式右边的值

语句

语句是控制程序分支运行的指令

条件语句

  • if语句: 基于不同的条件来执行不同的动作, 根据判断的条件的执行结果(true或false)来决定执行的代码块

  • 条件表达式

    语法: condition ? expression1 : expression2

    行为: 若condition为true, 返回expression1, 否则返回expression2

  • switch语句: 用于根据表达式的不同取值执行对应的代码块

循环语句

用于重复执行相同的一组语句, 提高效率, 简化代码.

  • for语句: 会被重复执行, 直到循环条件结果为false

  • for-of语句: 可遍历数组等可迭代的类型

  • while语句: 循环表达式是在执行循环体之前判断的

  • do-while语句: 循环控制表达式是在执行循环体之后判断的

break语句: 用于立即终止当前循环(如: for, while, do-while), 跳出循环体执行后续代码

continue语句

throw语句: 主动抛出异常

try语句: try-catch-finally包裹可能抛出异常的代码块, 捕获并处理异常

函数

函数是将多条语句组合在一起, 用于实现特定功能或完成特定任务的, 可重用的代码块.

通过function关键字声明要告诉编译器函数的名称, 返回类型和参数以及执行的内容.

function 函数名 (参数列表): 返回类型 { 函数体 }

参数类型:

  • 必选参数: 函数调用时必须要传入的参数
  • 可选参数: 参数的格式可为name?:Type, 表示参数是可选的, 即在调用函数时可以选择性传入的参数
  • 默认参数: 允许开发者为参数指定默认值, 在调用函数时若未传递相应的参数, 则使用默认值
  • 剩余参数: 函数的最后一个参数可以是剩余参数, 格式为...restArgs, 允许开发者接收数量不定的实参, 并将它们作为数组处理
参数类型 语法格式 作用 是否必传 使用场景
普通参数 参数名: 类型 接收指定类型的传入值 是 必须传入的基础参数
可选参数 参数名?: 类型 允许调用 时省略该 否 非必需参数
默认参数 参数名: 类型 = 默认值 未传参数时使用默认值 否 简化调用且需保底值的场景
剩余参数 …参数名: 类型[] 接收任意数量的同类型参数, 转为数组处理 否 不定数量参数

返回类型:

函数返回类型定义了函数执行完成后返回值的数据类型.

函数体内所有return语句的返回值类型必须与声明的返回类型严格匹配.

不需要返回值的函数的返回类型可以显式指定为void或省略标注.

函数的作用域

函数中定义的变量和其他实例仅可以在函数内部访问, 不能从外部访问
