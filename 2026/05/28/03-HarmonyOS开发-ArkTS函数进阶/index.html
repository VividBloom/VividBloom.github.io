HarmonyOS开发-ArkTS函数进阶

箭头函数/lambda表达式

箭头函数是一种简化函数定义的语法形式, 能够提升代码的简洁性

使用场景: 常用于快速编写简短函数, 尤其适合作为参数传入其他函数(如: 循环, 判断中的小逻辑)的场景, 语法简洁且无需多余关键字.

如果函数里只需要执行一行代码, 包裹代码的花括号可以省掉, 这行代码的结果自动当作返回值, 不用写retrun

函数类型

函数类型是用于约束函数参数类型和返回类型的数据类型

使用场景: 函数类型适用于把函数赋值给变量, 或作为参数传给其他函数时的场景, 能避免写错参数/返回值类型, 让代码更规范.

基础语法: type FuncType = (param1: 类型1, param2: 类型2) => 返回值类型

回调函数

回调函数: 把一个函数当成参数传给另一个函数, 接收参数的函数在 特定时机或条件下执行这个传入的函数

使用场景: 适合用在让一个函数完成核心逻辑后, 自动执行一在段可灵活替换的自定义逻辑的场景

闭包函数

闭包函数: 由函数及声明该函数的环境组合而成. 该环境包含了这个闭包创建时作用域内的任何局部变量

使用场景: 适用于需要让函数记住并保留自己定义时的变量(比如: 计数), 即使函数在其他地方被调用的场景

函数重载

函数重载是一种允许开发者通过不同参数类型或数量定义同名函数的机制

使用方法: 需先声明多个无函数体的重载签名, 再通过一个兼容所有签名的实现函数完成具体逻辑

不允许重载函数有相同的名字和参数列表, 否则将导致编译错误
